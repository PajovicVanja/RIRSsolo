Project Description

This project is a comprehensive vehicle management system designed to streamline the process of reserving, managing, and maintaining company vehicles. The system includes features for user authentication, vehicle reservation, issue reporting, and more. It supports multiple user roles, including drivers, managers, and administrators, each with specific functionalities tailored to their needs.

The frontend of the application is built using React.js, providing a responsive and user-friendly interface. The backend is powered by Node.js and Express, with Firebase Firestore as the database for storing vehicle and reservation data. The application also integrates with Cloudinary for handling image uploads, such as driver's license images.

Key features of the project include:

  • User authentication and role management
  • Vehicle reservation and management
  • Issue reporting and tracking
  • Admin dashboard for managing reservations and vehicles
  • Responsive design for optimal user experience on various devices
